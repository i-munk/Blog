📌 블록체인 개요

블록체인은 데이터를 블록 단위로 저장하고 이를 체인 형태로 연결하여 분산된 네트워크에서 관리하는 기술입니다. 중앙기관 없이 참여자들이 공동으로 데이터를 검증하고 저장하는 분산 원장 기술(DLT)의 한 형태입니다.

📌 블록체인의 주요 특징

  • 탈중앙화: 중앙 기관 없이 P2P 네트워크로 운영
  • 변조 불가능성: 데이터를 위조하거나 수정할 수 없음
  • 보안성: 암호화 기술을 활용해 높은 신뢰성 보장
  • 투명성: 누구나 거래내역을 확인할 수 있음
  • 스마트 컨트랙트: 자동화된 계약 기능 제공

📌 블록체인 기술의 기원과 등장 이유

🔹 신뢰 문제 해결

기존 금융 시스템에서는 중앙기관(은행, 정부)이 신뢰를 보장하지만, 높은 수수료, 거래 지연, 데이터 조작 가능성 등의 문제가 있었습니다. 이를 해결하기 위해 블록체인이 등장하였습니다.

🔹 비트코인의 등장

2008년, 사토시 나카모토가 비트코인 백서를 발표하면서 블록체인의 개념이 구체화되었습니다.

  • 중앙기관 없이도 개인 간(P2P) 송금 가능
  • 이중 지불(Double Spending) 문제 해결
  • 높은 수수료와 거래 속도 문제 개선

📌 타임스탬프란?

타임스탬프는 데이터가 생성되거나 변경된 시간 정보를 기록하는 것을 의미합니다.

🔹 주요 기능

  • 데이터 무결성 보장 (변조 방지)
  • 순서 보장 (이전 기록과의 관계 유지)
  • 보안성 강화 (전자 문서, 금융 거래 인증)
  • 이력 관리 (로그 데이터 저장)
