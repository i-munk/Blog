📌 이더리움 (Ethereum)

스마트 컨트랙트(Smart Contract)와 분산 애플리케이션(DApp)을 실행할 수 있는 확장 가능한 블록체인 플랫폼입니다.

🔹 창시자: 비탈릭 부테린 (Vitalik Buterin)

  • 비트코인에 튜링 완전성을 추가하자는 제안을 했으나 거절당함
  • 프로그래밍 가능한 블록체인 이더리움을 개발

📌 이더리움의 핵심 기술

  • 스마트 컨트랙트 (Smart Contract): 블록체인에서 실행되는 자동화된 계약
  • EVM (Ethereum Virtual Machine): 모든 스마트 컨트랙트를 실행하는 가상 머신
  • PoS 합의 알고리즘: 기존 PoW에서 PoS로 전환하여 확장성과 보안 강화

📌 이더리움의 블록체인 모델

이더리움은 어카운트 기반 모델 (Account-based)을 사용합니다.

  • 모든 사용자와 스마트 계약이 계정(Account)으로 표현됨
  • 계정을 통해 상태(State)와 자산 관리

📌 계정 종류

  • 외부 소유 계정(EOA, Externally Owned Account): 개인이 소유한 계정으로, 개인키로 제어됨
  • 스마트 계약 계정(Contract Account): 스마트 컨트랙트가 배포된 계정으로, 트랜잭션 발생 시 코드 실행

📌 계정 상태

  • Nonce: 해당 계정에서 보낸 트랜잭션 수
  • Balance: 계정에 저장된 이더(Ether)의 양
  • Storage: 스마트 계약 데이터 (스마트 계약 계정에만 해당)
  • CodeHash: 스마트 계약 코드 (스마트 계약 계정에만 해당)

📌 트랜잭션 방식

이더리움에서는 계정 간에 트랜잭션을 보내 상태를 변경할 수 있습니다.

  • 이더(Ether)를 전송하는 트랜잭션
  • 스마트 컨트랙트를 호출하는 트랜잭션

📌 비트코인 VS 이더리움

기준 비트코인 이더리움
목적 디지털 화폐 (P2P 전자 화폐, 디지털 골드) 탈중앙화된 글로벌 컴퓨터 (스마트 컨트랙트 실행)
확장성 제한적 (확장성 문제 존재) 스마트 컨트랙트 및 DApp 지원
합의 알고리즘 PoW (작업 증명) PoS (지분 증명, 이더리움 2.0)
네트워크 수수료 거래 수수료 (Bitcoin Fee) 가스(Gas) 시스템을 이용한 수수료

📌 스마트 컨트랙트 (Smart Contract)

블록체인에서 실행되는 자동화된 계약(프로그램)으로, 특정 조건이 충족되면 자동으로 실행됩니다.

🔹 특징

  • 자동 실행: 계약 조건이 충족되면 즉시 실행됨
  • 변경 불가능 (Immutable): 배포 후 코드 변경 불가능 (업그레이드는 가능)
  • 투명성: 블록체인에 기록되므로 누구나 검증 가능
  • 보안성: 중앙 서버 없이 실행되므로 해킹 위험 감소
